OBJECTIVE:-

1)The aim of this project is to analyze the impact of social media usage on students’ lifestyle and academic performance.

2)It focuses on identifying patterns between screen time, sleep hours, addiction levels, and mental health across different age groups and education levels.

3)The project aims to uncover how digital behavior influences student well-being and productivity.

INSIGHTS:-

1)Usage trends across age show fluctuations, with peak usage in late teens/early 20s, suggesting critical intervention stages.

2)Sleep patterns vary with age, and reduced sleep is observed alongside higher usage hours.

3)Gender distribution is almost equal, indicating that addiction impact is not gender-specific.

4)The dataset includes 705 students, with an average of 4.92 hours of daily usage and 6.87 hours of sleep, indicating a moderate imbalance.

5)Around 64% of students are affected academically, showing a strong link between high usage and reduced academic performance.

6)The dashboard enables interactive filtering (age, gender, country), helping uncover deeper behavioral insights.

7)The project demonstrates how data visualization can reveal hidden patterns in student habits and mental health.

CONCLUSION:-
1)This project highlights the growing concern of social media addiction among students.

2)It provides data-driven insights that can help educators and policymakers take informed actions.
